# C64Imager
C64 Imager is a small tool to convert JPG images into C64 graphics. The output is provided both as a BMP and a PRG file to view the image on a C64 or in an emulator.

Copy the repository to your computer. Extract it into a folder. If necessary, install Python (3.14) on your computer. Then first run Requirements.bat and afterwards Make.bat. After that, you will find the EXE file in the dist folder. Have fun!
